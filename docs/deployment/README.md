# Deployment Documentation

This directory contains guides for deploying and maintaining SurfSense in production environments.

## Contents

- **DEPLOYMENT_GUIDE.md** - Complete deployment guide including VPS setup and common issues
- **LOCAL_LLM_SETUP.md** - Installing and configuring Ollama with local models
- **NGINX_CONFIGURATION.md** - Nginx setup and session handling
- **LLM_CONFIGURATION.md** - Configuring LLM providers and fallback system
- **OPTIMIZATION_GUIDE.md** - Performance optimization strategies
- **GIT_WORKFLOW.md** - Git workflow for pushing changes to production

## Quick Start

For first-time deployment, start with DEPLOYMENT_GUIDE.md.
